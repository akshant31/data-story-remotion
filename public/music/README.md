# Background music

Drop a track in `public/music/`, then in the spreadsheet's **Music** column put
`music/yourtrack.mp3` (one cell per reel is enough — or paste a URL). The track
**loops, fades in/out, and automatically ducks** to ~45% volume while the
voiceover is speaking, then swells back in the gaps. Tune loudness with the
**Music Vol** column (0.12 quiet … 0.25 present; default 0.18).

## Where to get tracks (royalty-free, monetization-safe)

Free:
- YouTube Audio Library — youtube.com/audiolibrary (free, download as files)
- Pixabay Music — pixabay.com/music
- Bensound — bensound.com (free with credit, or paid licence)
- Uppbeat — uppbeat.io (generous free tier with credits)

Paid subscriptions (cheap, cross-platform safe):
- Uppbeat+ — ~$6/mo, best budget pick for Reels/Shorts
- Epidemic Sound — ~$10–15/mo, huge library + stems, pre-cleared for YouTube
- Artlist — ~$10–17/mo, Universal licence, includes AI music + SFX

AI-generated (custom track per topic, commercially safe):
- ElevenLabs Music — built on licensed data, API-ready (you already use Sarvam
  for voice; this pairs well)
- Udio / Suno — strong quality; check the current commercial-use terms
- Stable Audio — instrumental beds, open-weight option

## Picking a vibe per topic

- AI reels → tense / driving electronic
- Football & cricket → energetic, percussive sport
- Chess → minimal, cinematic, sparse piano

Pick low-energy, loopable instrumentals without vocals — vocals fight your
voiceover. Aim for tracks that don't peak hard so the duck stays smooth.
